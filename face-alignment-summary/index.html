Face Alignment Summary

Published at Aug 24, 2016 • Face Alignment

以我目前所读论文为依据，现有算法大体上分为两种类型：

  • 1. classifying search windows
  • 2. directly predicting key point positions(or shape parameters)

暂时没有接触第一种category的paper，第二种还可以细分为两种：

  • 1. 基于模型(Model Based)，代表有AAM、ASM
  • 2. 基于回归(Regression Based)。还可以细分为基于线性回归的，代表有CPR；基于非线性回归的，代表有DCNN、CFAN、TDA

对每个paper需要关注这几个点：

  • 框架或模型：基于model的框架，cascaded（级联）框架，deep learning框架等等
  • 优缺点：算法的speed、accuracy、是否利用geometric constraints，额外因素如occlusion、pose variations、lighting的影响
  • Feature是如何提取的

Model Based

基于模型的论文都对人脸建立了一定的模型，比如：AAM模型、ASM模型、CLM模型。论文认为模型可以刻画出人脸图像，所以只需要求得特定人脸的模型参数，即可合成对应的人脸图像

基于模型的方法难点在于人脸模型的建立，而且对初始值很敏感，鲁棒性不高

这方面论文看了4篇，理解多，简略介绍

Regression Based

该方法尝试学习 从deteced face region 到 facial landmarks的 mapping，有用线性和非线性regressor这两种

1.线性回归

1.1 Cascaded Pose Regression（CVPR 10）

级联回归模型在CVPR 10首次提出。它的核心思想是学习多个回归器，每个回归器预测当前当前形状和真是形状之间的偏差，然后将偏差合成到当前形状，形成新的预测形状，作为下一个回归器的输入，从而逐步逼近真实值。每个回归器均不一样，需要预先提取特征，作为回归器的输入，特征采用pose-indexed features（不是很理解特征提取这方面，懂的人可以告知本人，不胜感激），回归函数采用Random Fern Regressors

CPR有明显的缺点：其对初始化形状非常敏感，论文中使用K次不同的初始化形状来测试，并融合K次的测试结果作为最终预测形状。而且CPR不能有效解决occlusion问题

CPR出现后，线性回归模型大都基于级联模型，统一称作级联回归模型

1.2 Face Alignment by Expicit Shape Regression（CVPR 12）

论文设计了two-level boosted regression，使用correlation-based feature selection方案来提取shape indexed feature

Two-Level Boosted Regression具体是指：External-level和Internal-level

其中最原始的回归器r基于random fern学习

1.3 Using Conditional Regression Forests（CVPR 12）

论文不同于基于Regression Forest的方法，它的核心是Conditional Regression Forests。基于Regression Forest学习到facial image patches和location of feature points之间的mapping，而conditional regression forest则基于face image的某些属性预先进行分类，然后基于特定分类下的regression forest学习相应的mapping

1.4 Robust Face Landmark estimation under occlusion（ICCV 13）

又名：Robust Cascaded Pose Regression（RCPR），旨在解决CPR的遮挡问题，它在预测landmarks的同时预测特征点被遮挡的状态

同时RCPR对CPR的初始化问题也提出了trick，即智能重启：随机初始化一组形状，运行至级联模型的前10%回归器，统计形状预测的方差，如果方差小于一定阈值，说明这组初始化不错，则跑完剩下的90%的级联函数，得到最终的预测结果；如果方差大于一定阈值，则说明初始化不理想，选择重新初始化一组形状

1.5 DRMF（CVPR 13）

论文首先用Constrained Local Models（由之前某篇论文提出，与AAM、ASM类似）对人脸进行建模。之后DRMF利用SVR对回归器建模，并使用HOG特征（相对的有：SIFT特征）最为回归函数的输入，最终预测出CLM的人脸模型参数，而不是直接预测人脸的形状。由于模型很难刻画出完整的人脸变化形状，所以鲁棒性欠缺

1.6 Regressing Local Binary Features（CVPR 14）

LBF的亮点是二值化特征的提取，提取遵循论文所说的局部原则（locality priciple）：

  • 高可判的纹理信息大多落在当前预测特征点（由级联模型的前一步得到）周围的局部区域
  • 形状上下文信息（比如其他特征点的位置）和该特征点的局部纹理都提供了充足的信息

具体的来说，LBF基于random forest在局部区域学习局部稀疏二值化特征，因此计算开销大大减少

1.7 Face Alignment with an Ensemble of Regression Trees（CVPR 14）

论文基于回归树学习级联模型中的回归函数

2.非线性回归：利用CNN进行非线性建模

2.1 CUHK的DCNN（CVPR 13）提出了3层框架

第一层使用整个face image作为input region（所以特征是global high-level feature），得到了initial predicitons，所以也被称为initialization stage。而且同时进行多个不同的key points的预测，所以能有效利用geometric constraints

之后的二三层对第一层的结果进行refine，每层的cnn以当前预测点为中心，提取local feature，预测出当前facial points和ground truth之间的偏差

可以看出DCNN采用coarse-to-fine的步骤，一步一步逼近ground truth，因此也属于cascaded（级联）模型

2.2 CFAN（ECCV 14）

与DCNN类似，也是coarse-to-fine的级联模型，第一个CNN预测出初始值，后面的CNN用于refine并逼近真实值。DCNN二三层是对第一层初始结果的refine，使用没有shape constraint的local features，因此容易得到局部最优值，对occlusion等影响因素的鲁棒性不理想。CFAN解决了这个问题，在同样的第一层处理之后，之后的每层使用point之间的约束信息（constraint），因此CFAN对第一步init值的敏感度要比DCNN低一些。另外，CFAN的每层的input image的分辨率也只逐渐变高的，第一步用于快速的预测初始值，低分辨率的图片对应于large search step，后面紧跟的refine则选择使用逐渐变高的分辨率进行

2.3 TDA（ACCV 14）

TDA与DCNN、CFAN大体类似，其不同之处是先根据Face Image的Topic进行分类，然后使用每个分类下独立特有的CNN网络进行处理

这里的Topic分类是指通过k-means clustering对人脸图片进行分类，来决定人脸图片属于5中Topic中的哪一类，以决定之后使用某具体分类下的CNN网络

可以看出上面三种算法都将Image To Shape的mapping分解到级联框架的各个stage分别进行训练或处理，而且利用CNN可以免于特征的手动提取

PS：CFAN和TDA是由中科院张杰博士于同一年提出👍
